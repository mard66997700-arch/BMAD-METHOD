**Why three surfaces, not one:**

| Use case | Best surface | Why |
|---|---|---|
| Conversation outdoors with a stranger | Mobile app | Mic input, OS STT free, no laptop |
| Travel — read foreign menus, ask directions | Mobile app | Same; small form factor |
| Watching foreign YouTube on PC | Chrome extension (Phase A) | Captions are already authored, lowest latency |
| Watching Netflix / Twitch / podcast on PC | Chrome extension (Phase B) | No captions API → local Whisper |
| Live meeting on PC (Zoom / Meet) | Chrome extension Phase B + Whisper | Tab capture + local STT |

## 4. What is solid (don't redo)

1. **Audio pipeline** in `app/src/core/audio/` — capture → high-pass
   → VAD → chunker. 41 test files, 100% deterministic, no real audio
   needed.
2. **Engine routers** — STT / Translation / TTS each have an
   abstract router with mock + real providers; engine-router.test.ts
   covers dual-ear panning, multi-utterance, language switch.
3. **Free Google translate endpoint** — used by both `app/` and
   `extension/`, kept in lock-step (same response-shape parser).
4. **YouTube captions parser** — pure functions
   (`extractPlayerResponse`, `pickCaptionTrack`, `parseJson3Events`,
   `findEventAt` binary search), 30 unit tests, verified end-to-end
   against realistic fixture + live Google translate.
5. **Whisper-WASM wrapper** — factory-injected so the real model is
   never loaded in tests; AbortSignal-aware; ISO-639-1 → Whisper
   language-name map.
6. **Manifest & permissions** — only requests `tabCapture`,
   `offscreen`, `storage`, `activeTab`, `scripting`. **No mic
   permission required** for YouTube tab capture.

## 7. Risk register (updates only)

These are new risks introduced by the zero-key path. Existing risks
in `_bmad-output/architecture.md` still apply.

| ID | Risk | Likelihood | Impact | Mitigation |
|---|---|---|---|---|
| R-EXT-01 | YouTube blocks `timedtext` from extension origin | Med | High (Phase A breaks) | Detect 403/empty response → auto-fallback to Whisper-WASM; surface "Captions blocked" status |
| R-EXT-02 | Hugging Face rate-limits the model fetch | Low | High (Phase B first-load broken) | Self-host on Cloudflare R2; add SHA-256 integrity check |
| R-EXT-03 | Whisper-tiny accuracy too low for Japanese / Korean | Med | Med (user trust drops) | Per-language gating; suggest paid Whisper API in popup when source is JA/KO |
| R-EXT-04 | esbuild bundle (`dist/`) drifts from source | Low | Med (silent failure on load-unpacked) | Pre-commit script: refuse commit if `dist/` exists in working tree (already in `.gitignore`); `npm run build` instructions in TESTING.md |
| R-EXT-05 | YouTube changes `ytInitialPlayerResponse` JSON shape | Low | High (parser breaks) | Pure-function parser already covers null / empty / missing fields; integration smoke test in CI once enabled |
